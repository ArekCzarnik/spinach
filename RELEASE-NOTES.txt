              spinach 0.1 RELEASE NOTES

This is the initial spinach release with basic support.

Features
------------
* Sync and Async API
* SSL Support (over stunnel)
* Unix domain sockets (linux-x86_64 only)
* Cluster-Support (connect to one node with the ability to provide a list of nodes)

spinach requires a minimum of Java 8 to build and Java 6 run. It is tested continuously against Disque.
